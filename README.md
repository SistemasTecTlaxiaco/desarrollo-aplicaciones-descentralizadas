## Roles and Teams

## Evaluation System
- **Story Points**: Measure the effort required to complete tasks; higher points indicate more complex tasks.
- **Velocity**: Amount of work the team can complete in a sprint, calculated by the total story points completed.

## Technologies to be Used
- **Stellar**: Blockchain platform for dApp development.
- **ICP**: Internet Computer for decentralized applications.
- **Kahoot**: For interactive learning sessions.
- **GitHub Projects**: For project management.
- **eduScrum**: Agile framework for educational purposes.

## Learning Objectives
- Understand the architecture of Web3.
- Compare different consensus mechanisms utilized in blockchain technologies.
- Demonstrate transactions on the blockchain.
